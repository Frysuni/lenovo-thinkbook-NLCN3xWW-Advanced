# 8. Вход в UEFI Shell (Linux и Windows)

`BiosAdvancedPatch.efi` и ваш `SetupUtilityApp.efi` запускаются из **UEFI
Shell**. Эта страница показывает, как туда попасть из любой ОС.

> Клавиши ThinkBook (G6+ AHP / 21LG): **F2** (или **Fn+F2**) — вход в BIOS Setup,
> **F12** (или **Fn+F12**) — одноразовое меню загрузки. Модификатор `Fn` нужен,
> если «Hotkey Mode / FnLock» заставляет ряд F-клавиш работать как медиаклавиши.

## 0. Разовая подготовка (обе ОС)

1. **USB-флешка с FAT32.** Положите в корень два файла:
   - `BiosAdvancedPatch.efi` (этот проект);
   - `SetupUtilityApp.efi` (извлечён из *вашей* прошивки — см.
     [сборку и запуск](04-build-and-run.md)).
2. **UEFI Shell для загрузки.** Два варианта:
   - Многие прошивки Insyde его уже содержат — ищите пункт **«UEFI Shell»** или
     **«EFI Shell»** в меню загрузки F12. Если он есть, добавлять ничего не нужно.
   - Иначе добавьте стандартный шелл на флешку: скачайте `shellx64.efi` (UEFI
     Shell из EDK2, например из релизов tianocore/edk2) и сохраните как
     **`\EFI\BOOT\BOOTX64.EFI`** на флешке. Прошивка предложит флешку как
     загрузочное UEFI-устройство.
3. **Secure Boot** обычно нужно **выключить**, чтобы запускать эти неподписанные
   EFI-файлы. В BIOS Setup (**F2**) → **Security → Secure Boot → Disabled**,
   сохраните и выйдите.

## 1. Из Windows

### Подготовить флешку

- Отформатируйте USB в **FAT32** (правый клик → Форматировать, или `diskpart`).
- Скопируйте `BiosAdvancedPatch.efi` и `SetupUtilityApp.efi` в корень.
- Если нужен шелл: скопируйте `shellx64.efi` в `\EFI\BOOT\BOOTX64.EFI` на флешке
  (создайте папки). [Rufus](https://rufus.ie) тоже умеет записать образ UEFI
  Shell, но достаточно обычной FAT32-флешки с `BOOTX64.EFI`.

### Загрузиться в него

Любым способом:

- **Меню загрузки:** вставьте флешку, перезагрузитесь и нажимайте **F12** (или
  **Fn+F12**) на логотипе Lenovo. Выберите флешку / «UEFI Shell».
- **«Особые варианты загрузки» Windows:** Параметры → Система → Восстановление →
  *Перезагрузить сейчас* в разделе особых вариантов → **Использовать устройство**
  → ваша флешка. (Или `shutdown /r /o /t 0` и выберите *Использовать
  устройство*.)

Если в Windows включён **BitLocker**, приостановите его перед изменением настроек
BIOS/Secure Boot, чтобы избежать запроса ключа восстановления:
`manage-bde -protectors -disable C:`.

## 2. Из Linux

### Подготовить флешку

```bash
# Определите флешку (например, /dev/sdX) через lsblk — точно выберите USB!
sudo mkfs.vfat -F32 /dev/sdX1            # форматировать в FAT32 (стирает её)
sudo mount /dev/sdX1 /mnt
sudo cp BiosAdvancedPatch.efi SetupUtilityApp.efi /mnt/
# Опционально: добавить шелл, чтобы флешка сама была загрузочной
sudo mkdir -p /mnt/EFI/BOOT
sudo cp shellx64.efi /mnt/EFI/BOOT/BOOTX64.EFI
sudo umount /mnt
```

### Загрузиться в него

- **Меню загрузки:** перезагрузитесь и нажимайте **F12** (или **Fn+F12**) на
  логотипе Lenovo, затем выберите флешку / «UEFI Shell».
- **Или добавьте загрузочную запись** через `efibootmgr` (продвинуто):
  ```bash
  sudo efibootmgr -c -d /dev/sdX -p 1 \
       -L "UEFI Shell" -l '\EFI\BOOT\BOOTX64.EFI'
  ```
  затем выберите её в меню загрузки прошивки.

## 3. В шелле — найти флешку и запустить

Шелл показывает файловые системы как `FS0:`, `FS1:`, … Сопоставьте и переключитесь:

```
Shell> map -r                 # обновить / показать файловые системы
Shell> fs0:                   # пробуйте fs0; если это не флешка — fs1:, fs2: …
FS0:\> ls                     # убедитесь, что здесь виден BiosAdvancedPatch.efi
FS0:\> connect -r             # убедиться, что драйвер браузера форм загружен
FS0:\> BiosAdvancedPatch.efi  # применить рантайм-патч (ждите "OK (verified)")
FS0:\> SetupUtilityApp.efi    # запустить штатную настройку — вкладка Advanced на месте
```

Если `fs0:` — не то устройство, выполняйте `ls` после каждого `fsN:`, пока не
увидите `BiosAdvancedPatch.efi`. Что означает вывод патчера — см.
[сборку и запуск](04-build-and-run.md#диагностика--читайте-вывод-приложения).

## 4. Заметки

- Патч действует на одну загрузку. После любого сброса снова входите в шелл и
  снова запускайте `BiosAdvancedPatch.efi`.
- При желании верните Secure Boot включённым для обычной работы; сама
  разблокировка не оставляет постоянных изменений (но любые настройки BIOS,
  которые вы измените в Advanced, **сохраняются** — см.
  [безопасность](06-safety-and-limitations.md)).

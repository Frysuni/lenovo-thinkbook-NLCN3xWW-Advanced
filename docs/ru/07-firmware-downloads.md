# 7. Загрузка прошивок

> Прошивки в репозитории нет (копирайт Lenovo/Insyde). Скачайте пакет BIOS для
> своей машины у Lenovo. Ссылки ниже — отправные точки; Lenovo меняет реальные
> URL файлов и закрывает их недолговечными токенами, поэтому всегда переходите со
> страницы официальной поддержки, а не используйте захардкоженную ссылку CDN.

## Идентификация модели

| Маркетинговое имя | Тип | Семейство BIOS |
|---|---|---|
| ThinkBook 16 G6+ AHP | 21LG | `NLCN` |
| ThinkBook 14 G6+ AHP | 21LF | `NLCN` |

Уточните точный номер типа (наклейка на дне, либо `wmic baseboard get product` в
Windows) перед загрузкой.

## Официальные страницы поддержки

G6+ AHP — преимущественно модель для рынка Китая; её BIOS раздаётся из Lenovo
China. Используйте поиск по сайту поддержки по **номеру типа** (например, `21LG`)
или имени модели:

- Поддержка Lenovo China (драйверы/BIOS):
  `https://newsupport.lenovo.com.cn/` — ищите `ThinkBook 16 G6+ AHP` или `21LG`,
  затем **驱动下载 / BIOS**.
- Глобальная поддержка Lenovo (та же схема URL, что у других ThinkBook):
  `https://pcsupport.lenovo.com/` — ищите номер типа, затем
  **Drivers & Software → BIOS/UEFI**.

Нужный пакет — это инсталлятор Insyde `H2OFFT` **«BIOS Update (Flash from
Operating System)»** с именем `BIOS-NLCNxxWW.exe` / `NLCNxxWW.exe`.

## Прямой шаблон CDN (только для справки — токены истекают)

Lenovo China раздаёт файлы с:

```
https://newdriverdl.lenovo.com.cn/newlenovo/alldriversupload/<package-id>/<file>?token=<...>
```

Наблюдавшиеся ID пакетов (сегмент `<package-id>`), для справки при сопоставлении
версии в списке поддержки:

| Версия | файл | package id |
|---|---|---|
| NLCN22WW | `BIOS-NLCN22WW.exe` | 123091 |
| NLCN27WW | `BIOS-NLCN27WW.exe` | 128072 |

Параметр `token` ограничен по времени; голый URL без свежего токена возвращает
ошибку. Рабочую ссылку получите кликом по загрузке на странице поддержки.

## Извлечение прошиваемого образа и EFI-приложений

`.exe` — это самораспаковщик Insyde `H2OFFT`. Внутри найдёте SPI-образ
(`*.bin` / `*.FD` / `*.cap`) и инструменты прошивки. Из SPI-образа:

- Извлеките **`H2OFormBrowserDxe`** (GUID `9E5DAEB4-4B91-4466-9EBE-81C7E4401E6D`)
  для анализа — см. [воспроизведение анализа](03-reproduce-analysis.md).
- Извлеките **`SetupUtilityApp`** (PE32-секцию DXE-приложения настройки), чтобы
  запускать рядом с патчером — см. [сборку и запуск](04-build-and-run.md).

Извлечение делается через UEFITool / `uefiextract` из dev-окружения Nix
(`nix develop`).

## О регионах и зеркалах

Сторонние зеркала драйверов существуют, но непроверены — предпочитайте
официальную страницу поддержки Lenovo и сверяйте строку версии BIOS. Локальные
копии для анализа остаются вне этого репозитория через
[`.gitignore`](../../.gitignore).
